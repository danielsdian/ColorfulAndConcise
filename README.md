🎨 Colorful and Concise Badges for Nuvio
A complete, well‑organized badge pack designed specifically for Nuvio, featuring a clean color hierarchy, consistent visual logic, and optimized regex patterns for accurate media detection.

This pack is ideal for users who want clarity, technical precision, and aesthetic consistency when displaying media badges inside Nuvio.

🌈 Features
✔ Consistent Color Hierarchy
Every badge follows a structured color system:

Color	Meaning
🟨 Gold	Premium / top‑tier
🟩 Green	Very good / modern
🔵 Blue	Good / intermediate
🟧 Orange	Weak / borderline
🟥 Red	Poor / legacy / undesirable


This hierarchy is applied uniformly across all categories.

🧩 Badge Categories
Resolution
From 144p to 4K, with colors reflecting quality level.

Quality
Release source and type (Remux, BluRay, WEB-DL, HDRip, CAM, etc.).
Regex patterns are optimized to avoid overlap.

Visual
Image technologies such as Dolby Vision, HDR10+, HLG, IMAX, SDR, and more.
Includes a dedicated IMAX group for clarity.

Audio
Accurate detection of audio formats:
Atmos, DTS:X, TrueHD, DTS-HD MA, FLAC, AAC, OPUS, etc.

Encoder
Codec identification: AV1, HEVC, AVC, XviD, DivX.

Channels
Optional badges for audio channel layouts (5.1, 7.1).

Streaming
Badges for streaming services such as Peacock, Netflix, and others.

📄 How to Use in Nuvio
Open Nuvio → Settings → General → Streams → Stream Badges

In the Badge JSON URL field, paste:

Código
https://raw.githubusercontent.com/danielsdian/ColorfulAndConcise/main/badges.json
Click Import

Nuvio will load all groups and badges automatically
